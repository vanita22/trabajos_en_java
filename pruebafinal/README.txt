****Programa con multiconexión de bases de datos****

- Programa hecho en IDE Netbeans, programas de bases de datos de: Access, MySQL, SQL Server y Postgres.
- Antes de ocupar este programa, tiene que tener instalados los programas de bases de datos mencionados anteriormente.
- Abrir la carpeta llamada dist.
- Dentro de esta carpeta presionar el jar llamado "pruebafinal".
- En la carpeta dist también están los drivers MySQL, SQL Server y Postgres.
- En la carpeta bd están todas las bases de datos para ser importadas en los distintos programas.
