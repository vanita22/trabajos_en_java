****Programa cálculo de remuneraciones****

- Programa hecho en Netbeans y MySQL.
- Abrir la carpeta llamada dist.
- Dentro de esta carpeta presionar el jar llamado "proyecto", aqui se abrira un menú con varias opciones.
- En la carpeta dist también están los drivers MySQL, Jfreechart y los JasperSoft.
- En la carpeta bd está el archivo de base de datos llamado "empleados" para importarlo en el administrador del MySQL.

* Dentro del programa esta la opción de "Generar reporte", aquí se ven en pdf los sueldos liquidos de cada trabajador, al presionar el
 botón abrir toma tiempo ejecutar el pdf según el computador.
